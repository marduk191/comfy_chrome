Workflows

How to set up workflows:

  • Add multiple workflows that you can choose from when sending images
  • Open ComfyUI and load or create the workflow you want to use
  • Click "Save (API Format)" to export your workflow as JSON
  • Copy the entire JSON content and paste it in the workflow form below
  • The extension will automatically find the LoadImage node and replace its input
  • Optionally, specify the exact node ID if you have multiple LoadImage nodes
